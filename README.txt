PORTAFOLIO DIGITAL DE KATE CROSTHWAITE

Cómo abrirlo:
1. Abre la carpeta del proyecto.
2. Haz doble clic en index.html, o usa la extensión Live Server de VS Code.

Cómo publicarlo en GitHub Pages:
1. Crea un repositorio público.
2. Sube index.html y las carpetas css, js y assets a la raíz del repositorio.
3. En GitHub entra en Settings > Pages.
4. Selecciona Deploy from a branch, rama main y carpeta /(root).
5. Guarda y espera a que aparezca el enlace público.
